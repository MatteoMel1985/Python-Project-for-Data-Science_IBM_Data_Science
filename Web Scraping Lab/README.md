<h1 align="center">Web Scraping Using BeautifulSoup</h1>  

This Jupyter Notebook introduces the fundamentals of web scraping with Python. It demonstrates how to parse HTML content with `BeautifulSoup`, navigate the structure of a web page, and extract information from tags, attributes, strings, child elements, parent elements, and sibling elements.

The notebook also explains how to use filtering methods such as `find_all()` and `find()` to locate specific HTML elements. It then applies these techniques to practical examples, including scraping links, image sources, colour codes, and structured data from HTML tables.

Finally, the notebook shows how scraped table data can be transformed into pandas DataFrames, both manually with `BeautifulSoup` and more directly with `pandas.read_html()`. Overall, it provides a practical introduction to collecting, extracting, and organising web-based data for further analysis.
