<h1 align="center">Python Project for Data Science</h1>  

The Jupyter Notebook in this repository demonstrates how to extract financial stock data using Python. It uses the `yfinance` library to retrieve company information, historical share prices, trading volume, and dividends, then organises the data with pandas and visualises selected trends with `Matplotlib`.
